矿车+
========================

在这个项目是在《矿车》项目中添加了脱轨恢复功能，让帕克适应更弯曲的轨道。

.. image:: img/block/minec.png


**提示**

.. image:: img/block/sp210512_171727.png

使用另一个 **创建函数** 块让帕克备份并从急转弯中恢复。
请注意，新的 **创建函数** 函数不返回任何值，而仅用于重新定向帕克。
